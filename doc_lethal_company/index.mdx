# Modpack Lethal Company

## I. Installer R2ModMan

Pour installer le modpack, il vous faut
<DownloadButton data={{url: "https://thunderstore.io/package/download/ebkr/r2modman/3.2.2/", important: true}} /> [**R2ModMan 3.2.2**](https://thunderstore.io/package/download/ebkr/r2modman/3.2.2/),
il s'agit d'un gestionnaire de mods.

:::tip Liste des versions
Si besoin, toutes les versions de R2ModMan sont disponibles [ici](https://thunderstore.io/package/ebkr/r2modman/versions/).
:::

## II. Télécharger le modpack

* Sur R2ModMan, recherchez et sélectionnez **Lethal Company**
* Cliquer sur "Importer un profil" et saisissez le code suivant : **`01986c72-325c-f0c2-1c6e-21a004b03bb1`**
* Vous pouvez désormais lancer le jeu moddé !

:::info En cas d'échec
Si l'importation du profil échoue, demandez-moi sur Discord et je vous donnerai un nouveau code.
:::
