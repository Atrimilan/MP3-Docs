# Serveur Litchi

<p className="mp3-gradient-effect">**Veuillez suivre les étapes suivantes pour installer le modpack et rejoindre le serveur moddé !**</p>

### 1. Installer le Modpack

* Téléchargez et exécutez
  <DownloadButton data={{ url: "https://maven.neoforged.net/releases/net/neoforged/neoforge/21.1.233/neoforge-21.1.233-installer.jar", important: true }} />{" "}<a href="https://maven.neoforged.net/releases/net/neoforged/neoforge/21.1.233/neoforge-21.1.233-installer.jar" target="_self" style={{ fontWeight: "bold" }} title="Télécharger NeoForge 21.1.233" aria-label="Télécharger NeoForge 21.1.233">NeoForge 21.1.233</a>
  (Minecraft 1.21.1)
* Téléchargez le Modpack <DownloadButton data={{url: "https://www.dropbox.com/scl/fi/sskswbwsixr1is5c9n6vu/Litchi_Modpack_v1.zip?rlkey=0fj57usye7y6d6dbp8r6v9303&st=73u8kc2z&dl=1", important: true, targetBlank: true}}/>{" "}<a href="https://www.dropbox.com/scl/fi/sskswbwsixr1is5c9n6vu/Litchi_Modpack_v1.zip?rlkey=0fj57usye7y6d6dbp8r6v9303&st=73u8kc2z&dl=1" target="_self" style={{ fontWeight: "bold" }} title="Litchi_Modpack_v1" aria-label="Litchi_Modpack_v1">Litchi_Modpack_v1.zip</a>
* Dézippez-le modpack, puis glissez tous les fichiers `.jar` dans `%appdata%/.minecraft/mods`

### 2. Optimiser NeoForge

* Démarrez le launcher Minecraft et allez dans **CONFIGURATIONS**
* Cliquez sur les **[...]** à côté du "NeoForge" tout juste installé, et sélectionnez **MODIFIER**
* Déroulez **PLUS D'OPTIONS** et remplacez tout **ARGUMENTS JVM** par ceci : `-Xms6G -Xmx6G -XX:+UseZGC`

### 3. Rejoindre le serveur
* Démarrez le jeu avec NeoForge
* Rejoignez le serveur à cette adresse : <span className="gold-gradient-effect">**`mp3.pixelfucker.com:25568`**</span>
