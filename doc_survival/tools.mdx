# Outils & commandes

## Pseudo coloré

Vous pouvez modifier la couleur de votre pseudo avec les commandes suivantes :

* Mettre votre pseudo en couleur unie :
    ```ps
    /nickcolor <color:#ff1199>
    ```
    > La couleur est encodée en hexadécimal, précédée d'un `#`.
* Mettre votre pseudo en dégradé :
    ```ps
    /nickcolor <gradient:#ffb23b:#ff1376>
    ```
    > Voici un générateur de dégradé : https://cssgradient.io/.<br/>
    > Notez que vous pouvez ajouter davantage de couleurs, en les séparant par des `:`.
* Réinitialiser la couleur de votre pseudo :
    ```ps
    /nonick
    ```
