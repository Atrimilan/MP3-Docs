# Règles du jeu

## I. Objectif du jeu
* Capturer les salles des coffres ennemie en restant dedans pendant `30 secondes`.
* Vous ne pouvez pas faire d'assaut avant le `Jour 6` (lorsque le PVP est activé).
* Vous ne pouvez pas poser d'eau et de blocs partout (détails ci-dessous).

## II. Liste des règles

| Règle | Configuration |
|---|---|
| Activation du PVP | `Jour 2` |
| Activation du Nether | `Jour 2` |
| Activation de l'End | `Jour 3` |
| Activation des TNT | `Jour 6` |
| Début de la capture de salle des coffres | `Jour 6` |
| Enderperle pour un assaut | ❌ |
| Poser des blocs dehors / dans les cavernes | ❌ |
| Poser des torches + des panneaux dehors | ✅ |
| Poser de l'eau chez l'ennemi / pour un assaut | ❌ |
| Poser de l'eau dehors / dans les cavernes | ✅ |
| Taux d'apparition de creepers chargés | `10%` |
| Chance d'obtenir une TNT par un creeper chargé | `50%` |
| Durée de la capture de la salle des coffres | `30 secondes` |
| Quantité de joueurs requis pour la capture | Toute l'équipe |
