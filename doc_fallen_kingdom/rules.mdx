# Détail des règles

_* **Dehors :** Partout en dehors de votre base (map + cavernes + bases ennemies)_<br/>
_* **Assaut :** Attaquer une base ennemie_<br/>
_* **Capture :** Capturer une salle des coffres ennemie_

| Règle | Configuration |
|---|---|
| Activation du PVP | `Jour 2` |
| Activation du Nether | `Jour 2` |
| Activation de l'End | `Jour 3` |
| Activation des TNT | `Jour 6` |
| Début de la capture de salle des coffres | `Jour 6` |
| Enderperle pour un assaut | ❌ |
| Poser des blocs dehors | ❌ |
| Poser de l'eau dehors | ❌ |
| Poser des torches et des panneaux dehors | ✅ |
| Taux d'apparition de creepers chargés | `10%` |
| Chance d'obtenir une TNT par un creeper chargé | `50%` |
| Dimension maximale de la salle des coffres | `20 blocs` |
| Temps de capture de la salle des coffres | `30 secondes` |
| Nombre de joueurs pour la capture | Toute l'équipe |
